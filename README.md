# نظام إدارة المخزون والمبيعات الذكي (Inventory Management System)

### وصف وفكرة المشروع
تطبيق ويب ديناميكي مصمم لإدارة وتتبع مخزون المنتجات ومراقبة مستويات الكميات والأسعار بشكل فوري داخل الشركات أو المحلات التجارية، مع توفير واجهة تحكم خاصة بالمسؤول لتحديث محتويات المخزن.

### المتطلبات المطبقة بناءً على وثيقة "المشروع النهائي (1).pdf":
- **لوحة التحكم:** لوحة تحكم ديناميكية كاملة (Dashboard).
- **العمليات الأساسية:** تطبيق الـ CRUD الكامل على المنتجات والمخزون.
- **إدارة الحسابات:** تسجيل دخول وخروج آمن مع إدارة الـ Sessions والـ Cookies.
- **توزيع الصلاحيات:** فصل صلاحيات المسؤول (Admin) عن الموظف العادي (Staff).
- **الأمان والوسائط:** تصفية المدخلات (Validation) مع إتاحة رفع صور المنتجات وعرضها بالموقع وتنسيق الواجهات بقالب Bootstrap.
